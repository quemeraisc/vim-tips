### A few useful things to have in .vimrc file :

Get vi to guess the type of file 
```
filetype plugin indent on
```
Correct tabulation is important
```
 set shiftwidth=4
 set softtabstop=4
 sest expandtab
 ``` 
 
 Know where you are in the file, using line numbering, and ruler
 ```
 set number
 set ruler
 ```
